### Color Palette:

```css
Light Theme:
- Primary: #0ea5e9 (Professional Blue)
- Background: #f8fafc (Soft White)
- Cards: #ffffff (Pure White)
- Text: #0f172a (Dark Slate)
- Borders: #e2e8f0 (Light Slate)

Dark Theme:
- Primary: #38bdf8 (Bright Sky Blue)
- Background: #0f172a (Deep Slate)
- Cards: #1e293b (Slate Cards)
- Text: #f1f5f9 (Light Text)
- Borders: #475569 (Slate Borders)
```

- **Font Family**: Inter (Headings), System Stack (Body)
- **Font Weights**: 300, 400, 500, 600, 700
- **Line Heights**: Optimized for readability
- **Letter Spacing**: Improved tracking for headings
